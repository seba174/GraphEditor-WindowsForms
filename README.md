# GraphEditor-WindowsForms
GraphEditor app made for university.
Technologies:
- C#
- Windows Forms

Usage:
- Left mouse button - add a vertice (when cursor is not over any vertice) or add edge between selected and checked vertice
- Right mouse button - check vertice (with cursor over vertice) or uncheck it
- Middle mouse button - move checked vertice
- Delete key - remove selected vertice (and all edges connected to it)
